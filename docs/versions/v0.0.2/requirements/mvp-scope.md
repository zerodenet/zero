# MVP 范围

`v0.0.2` 继续在 `v0.0.x` 线上推进，不扩协议面，重点补节点组能力。

## 本版先做

- `selector` 运行时切换
- `fallback` 出站组
- `urltest` 出站组

## 本版语义

### selector

- 保持现有配置形态不变
- 运行中允许把 `selector` 组切到另一个已定义成员
- 当前通过本地 `--status-listen` 控制入口完成

### fallback

- 按配置顺序尝试成员出站
- 前一个成员建连失败时，自动切到下一个
- 某个成员一旦建连成功，本次会话就固定使用该成员
- `block` 作为成员时，表示链路走到这里就直接阻断

### urltest

- 周期探测组成员到指定 `http://` 目标的可达性和延迟
- 选择探测成功且延迟更低的成员
- 首次探测前默认使用第一个成员
- 本轮全部失败时保留当前选择
- 当前只支持本地运行时后台探测，不提供更复杂的控制面

## 本版不做

- 更复杂的健康检查调度器
- 组嵌套组
- 远程节点订阅
